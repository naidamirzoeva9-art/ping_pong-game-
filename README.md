**Описание игры «Пинг-понг»**

«Пинг-понг» — это мультиплеерная игра, реализованная на языке программирования Python с использованием библиотеки Pygame. Проект представляет собой прототип, который можно найти в репозитории

**Суть игры:**
Игроки соревнуются, отбивая виртуальный мяч ракетками. Цель — забить мяч на сторону противника так, чтобы тот не смог его отбить.

**Механика игры:**
* **Два игрока:** каждый управляет своей ракеткой.
  * Игрок 1 контролирует **левую ракетку** с помощью клавиш на клавиатуре.
  * Игрок 2 контролирует **правую ракетку** с помощью клавиш на клавиатуре.
* **Мяч:** перемещается по игровому полю, отскакивая от стенок и ракеток. Скорость и траектория мяча зависят от силы и угла удара.
* **Очки:** начисляются, когда противник не успевает отбить мяч.
* **Победа:** определяется по набранному количеству очков (количество побед настраивается).

**Особенности:**
* **Мультиплеерный режим:** позволяет двум игрокам соревноваться на одном устройстве.
* **Простая, но увлекательная механика:** идеально подходит для знакомства с основами геймдева.
* **Графическое оформление:** благодаря библиотеке Pygame игра имеет визуальные элементы — ракетки, мяч, игровое поле.
* **Кастомизация:** можно доработать логику, добавить уровни сложности, звуковые эффекты или изменить дизайн.
